探究原子世界（华中科大 科英双） 5.7

  • 引力场：各处 ggg 的变化 ⇒ 监测地下水、石油
  • 引力波：时空涟漪

凝聚态物理

  • 石墨 ⇐高温\Leftarrow{高温}⇐高温 金刚石（天然钻石形状定）
  • λ=hp动量=hmv\lambda = \frac{h}{p_{\text{动量}}} = \frac{h}{mv}λ=p动量​h​=mvh​（电子波动性）；E=p22m=h22mλ2E = \frac{p^2}{2m} = \frac{h^2}{2m\lambda^2}E=2mp2​=2mλ2h2​
  • 液体中，电子波干涉；晶体中，电子波干涉（√）

解释导电性：允带是否有电子

  • 晶体管：\ce{Si}→全钢石结构，会共：饱和
    • \ce{Si}：\ce{P}（多一个电子，nnn-type）、\ce{B}（少一个电子，ppp-type）→ 二极管
    • 1948 贝尔：晶体管 → 2015 IBM 7nm 台积电 2nm；1956 TAN 硬 → 2007 法 2014 LED×7 RGB
  • 摩尔定律：① 脱利、电子跑；② 延续50年、2016失效

扫描隧道显微镜

  • 1981 IBM 发明；1986 诺奖
  • 类似盲人摸 → \ce{Si(111)}
  • 原理：电子量子隧穿；高度灵敏 ⇒ 电流变化大 ⇒ 可得原子性质；微观尺度运动：扫描管（0.5pm / 2cm = 0.1μm / 4000m）
  • 超低湿、强磁场、超真空（平均自由程 1000km）
  • 原子定构：电子波 → 原子操纵：摆出特定结构

应用

  1. 分子电子学（最小电子器件），最高集成密度；分子作晶体管
  2. 自旋电子学：超高密度信息存储
  3. 超导：完全抗磁（★）；核磁共振（医院、保低温）
    • 悬浮山（常温超导）：
      • 方向1：找材料（零下100℃目标）
      • 方向2：原理探
    • 原理：超导能隙，用扫描
  4. 量子比特：
    • 问题：与外界有影响；任意一点可存储
    • 5d：拓扑量子计算 → 杯子=环（有圆）；对材料细节不关心，只求本质
