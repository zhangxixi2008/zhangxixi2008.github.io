欧拉函数乘积公式

由第三课我们已知

φ(n)=n∏p∣n(1−1p)\boxed{\varphi(n) = n \prod_{p \mid n} \left(1 - \frac{1}{p}\right)} φ(n)=np∣n∏​(1−p1​)​

欧拉函数的积性我们也证明过：

  1. 构造剩余性

    • A={a∣1≤a≤m, gcd⁡(a,m)=1}A = \{a \mid 1 \le a \le m,\ \gcd(a,m) = 1\}A={a∣1≤a≤m, gcd(a,m)=1}，共 φ(m)\varphi(m)φ(m) 个；
    • B={b∣1≤b≤n, gcd⁡(b,n)=1}B = \{b \mid 1 \le b \le n,\ \gcd(b,n) = 1\}B={b∣1≤b≤n, gcd(b,n)=1}，共 φ(n)\varphi(n)φ(n) 个；
    • C={c∣1≤c≤mn, gcd⁡(c,mn)=1}C = \{c \mid 1 \le c \le mn,\ \gcd(c,mn) = 1\}C={c∣1≤c≤mn, gcd(c,mn)=1}，共 φ(mn)\varphi(mn)φ(mn) 个。

  2. 用中国剩余定理建立映射

    考虑：

    {x≡a(modm)x≡b(modn)(gcd⁡(m,n)=1)\begin{cases} x \equiv a \pmod{m} \\ x \equiv b \pmod{n} \end{cases} \quad (\gcd(m,n)=1) {x≡a(modm)x≡b(modn)​(gcd(m,n)=1)

    • 该方程组在模 mnmnmn 下有唯一解 x，且 1≤x≤mn1\le x\le mn1≤x≤mn

    • 对于不同的 (a,b)(a,b)(a,b) 给出了不同的解 xxx

    由此我们建立了一个映射：

    f:(a,b)↦x(modmn)f: (a,b) \mapsto x \pmod{mn} f:(a,b)↦x(modmn)

    因为 gcd⁡(a,m)=1,  gcd⁡(b,n)=1,  gcd⁡(m,n)=1\gcd(a,m)=1,\;\gcd(b,n)=1,\;\gcd(m,n)=1gcd(a,m)=1,gcd(b,n)=1,gcd(m,n)=1 ，故 gcd⁡(x,mn)=1\gcd(x,mn)=1gcd(x,mn)=1，因此 x∈Cx\in Cx∈C

  3. 证明是双射

    取 c∈Cc\in Cc∈C，令 a≡c(modm),  b≡c(modn)a\equiv c\pmod m,\;b\equiv c\pmod na≡c(modm),b≡c(modn)，则 gcd⁡(a,m)=1,  gcd⁡(b,n)=1\gcd(a,m)=1,\;\gcd(b,n)=1gcd(a,m)=1,gcd(b,n)=1，故 (a,b)∈A×B(a,b)\in A\times B(a,b)∈A×B

综上：

φ(mn)=φ(m)φ(n)(gcd⁡(m,n)=1)\boxed{\varphi(mn) = \varphi(m)\varphi(n) \quad (\gcd(m,n) = 1)} φ(mn)=φ(m)φ(n)(gcd(m,n)=1)​

除数求和等式

我们要证明：

∑d∣nφ(d)=n\sum_{d\mid n}\varphi(d)=n d∣n∑​φ(d)=n

等价类证明

考虑分数 1n,2n,…,nn\dfrac 1 n,\dfrac 2 n,\dots,\dfrac n nn1​,n2​,…,nn​

我们把这些分数约分成最简形式 ad\dfrac a dda​，则：

  • ddd 必然是 nnn 的因子
  • 对于每个 ddd，相应 aaa 的个数总共有 φ(d)\varphi(d)φ(d) 个

因此把这些最简分数合起来，正好是原先 nnn 个分数。

阶/生成元角度证明

考虑群 Zn\mathbb Z_nZn​（模 nnn 的加法群），对于任意 k∈{1,2,…,n}k\in\{1,2,\dots,n\}k∈{1,2,…,n}

  • 元素 kkk 在 Zn\mathbb Z_nZn​ 中的阶 是 ngcd⁡(k,n)\dfrac n {\gcd(k,n)}gcd(k,n)n​
  • 反过来，对于每个 ddd，阶恰好等于 ddd 的元素个数为 φ(d)\varphi(d)φ(d)

把这些元素的阶分类统计，总元素个数为 nnn。

可以这么理解 ：Zn\mathbb Z_nZn​ 中的 nnn 个元素的阶必然为 nnn 的因数 ddd，因此把每个 ddd 对应元素（φ(d)\varphi(d)φ(d)）相加就是 nnn.

互素数和公式

求 1 到 nnn 中与 nnn 互素的数的和，记作：

S(n)=∑1≤k≤ngcd⁡(k,n)=1kS(n)=\sum_{\substack{1 \le k \le n \\ \gcd(k,n)=1}} k S(n)=1≤k≤ngcd(k,n)=1​∑​k

关键在于对称性观察

若 gcd⁡(n,k)=1\gcd(n,k)=1gcd(n,k)=1，则 gcd⁡(n−k,n)=1\gcd(n-k,n)=1gcd(n−k,n)=1

因此与 nnn 互素的数总是成对出现的

且在 n>2n>2n>2 时，k≠n−kk\ne n - kk=n−k

同时有 k+(n−k)=nk+(n-k)=nk+(n−k)=n

所以在 n>2n>2n>2 时，答案为：

S(n)=nφ(n)2S(n)=\dfrac{n\varphi(n)}2 S(n)=2nφ(n)​

其中：

S(1)=12S(2)=1S(1)=\dfrac 1 2 \\ S(2)=1 S(1)=21​S(2)=1
