Summation factor

To solve this equation :

anTn=bnTn−1+cna_nT_n=b_nT_{n-1}+c_n an​Tn​=bn​Tn−1​+cn​

We have to construct sns_nsn​ that satisfied : snbn=sn−1an−1s_nb_n=s_{n-1}a_{n-1}sn​bn​=sn−1​an−1​ (1). And in here, sns_nsn​ is the summation factor.

so if we define Sn=snanTnS_n=s_na_nT_nSn​=sn​an​Tn​. And when we multiple sns_nsn​, thus the original expression becomes :

Sn=Sn−1+sncnS_n=S_{n-1}+s_nc_n Sn​=Sn−1​+sn​cn​

This can be solved easily.

Yet how can we find sns_nsn​ ?

From the definiton (1), we know that sn=sn−1⋅an−1bns_n=s_{n-1}\cdot \dfrac {a_{n-1}}{b_n}sn​=sn−1​⋅bn​an−1​​, therefore :

sn=k⋅an−1an−2⋯a1bnbn−1⋯b2s_n=k\cdot \dfrac{a_{n-1}a_{n-2}\cdots a_1}{b_nb_{n-1}\cdots b_2} sn​=k⋅bn​bn−1​⋯b2​an−1​an−2​⋯a1​​

Harmonic number

Definition :

Hn=∑k=1n1kH_n = \sum_{k=1}^n\dfrac 1 k Hn​=k=1∑n​k1​

The summation of harmonic number :

∑0≤k<nHk=nHk−n\sum_{0 \le k < n}H_k=nH_k-n 0≤k<n∑​Hk​=nHk​−n

Proof :

∑0≤k≤n∑1≤i≤k1i=∑i=1n∑k=in−11i=∑i=1nn−ii=∑i=1n(ni−1)=n∑i=1n1i−n=nHn−n\begin{align*} \sum_{0 \le k \le n} \sum_{1 \le i \le k} \frac{1}{i} &= \sum_{i=1}^{n} \sum_{k=i}^{n-1} \frac{1}{i} \\ &= \sum_{i=1}^{n} \frac{n-i}{i} = \sum_{i=1}^{n} \left( \frac{n}{i} - 1 \right) \\ &= n \sum_{i=1}^{n} \frac{1}{i} - n = n H_n - n \end{align*} 0≤k≤n∑​1≤i≤k∑​i1​​=i=1∑n​k=i∑n−1​i1​=i=1∑n​in−i​=i=1∑n​(in​−1)=ni=1∑n​i1​−n=nHn​−n​

The methods are multiple, any ways you choose can get the answer easily.

Summing the numbers along rows or columns proves difficult, yet calculating along the diagnonals is straightforward.

Perturbation method

Suppose we are given the sequence term ana_nan​, and we aim to compute its partial sum SnS_nSn​, defined as:

Sn=∑k=0nakS_n=\sum_{k=0}^na_k Sn​=k=0∑n​ak​

Adding an+1a_{n+1}an+1​ to both sides, we derive the following identity step by step :

Sn+an+1=a0+∑k=0nak+1S_n+a_{n+1}=a_0+\sum_{k=0}^na_{k+1} Sn​+an+1​=a0​+k=0∑n​ak+1​

If we possess a recurrence relationship linking aka_kak​ and ak+1a_{k+1}ak+1​, the summation ∑k=0nak+1\sum_{k=0}^na_{k+1}∑k=0n​ak+1​ may be rewritten directly in terms of SnS_nSn​.

Take the geometric partial sum as an example : Sn=∑0≤k≤ncxkS_n=\sum_{0\le k\le n}cx^kSn​=∑0≤k≤n​cxk

We obtain the equation :

Sn+cxn+1=c+xSnS_n+cx^{n+1}=c+xS_n Sn​+cxn+1=c+xSn​

This is a linear equation containing only the unknown SnS_nSn​

Chebyshev’s monotonic inequality

(∑k=1nak)(∑k=1nbk)=n∑k=1nakbk−∑1≤j<k≤n(ak−aj)(bk−bj)\left( \sum_{k=1}^n a_k \right) \left( \sum_{k=1}^n b_k \right) = n \sum_{k=1}^n a_k b_k - \sum_{1 \le j < k \le n} (a_k - a_j)(b_k - b_j) (k=1∑n​ak​)(k=1∑n​bk​)=nk=1∑n​ak​bk​−1≤j<k≤n∑​(ak​−aj​)(bk​−bj​)

Therefore, if a1≤⋯≤an, b1≤⋯≤bna_1 \le \dots \le a_n,\ b_1 \le \dots \le b_na1​≤⋯≤an​, b1​≤⋯≤bn​，we have：

(∑k=1nak)(∑k=1nbk)≤n∑k=1nakbk\left( \sum_{k=1}^n a_k \right) \left( \sum_{k=1}^n b_k \right) \le n \sum_{k=1}^n a_k b_k (k=1∑n​ak​)(k=1∑n​bk​)≤nk=1∑n​ak​bk​

This is called the Chebyshev’s monitonic inequality

But how can we get the first equation ?

Define the sum :

S=∑1≤j<k≤n(ak−aj)(bk−bj)S = \sum_{1 \le j < k \le n} (a_k - a_j)(b_k - b_j) S=1≤j<k≤n∑​(ak​−aj​)(bk​−bj​)

Swap indices jjj and kkk:

S=∑1≤k<j≤n(aj−ak)(bj−bk)=∑1≤k<j≤n(ak−aj)(bk−bj)\begin{aligned} S &= \sum_{1 \le k < j \le n} (a_j - a_k)(b_j - b_k) \\ &= \sum_{1 \le k < j \le n} (a_k - a_j)(b_k - b_j) \end{aligned} S​=1≤k<j≤n∑​(aj​−ak​)(bj​−bk​)=1≤k<j≤n∑​(ak​−aj​)(bk​−bj​)​

Therefore:

2S=∑1≤j,k≤n(aj−ak)(bj−bk)−∑1≤j=k≤n(aj−ak)(bj−bk)2S = \sum_{1 \le j,k \le n} (a_j - a_k)(b_j - b_k) - \sum_{1 \le j = k \le n} (a_j - a_k)(b_j - b_k) 2S=1≤j,k≤n∑​(aj​−ak​)(bj​−bk​)−1≤j=k≤n∑​(aj​−ak​)(bj​−bk​)

The second term is 000.

The first term is:

∑1≤j,k≤najbj\sum_{1\le j,k\le n} a_j b_j 1≤j,k≤n∑​aj​bj​

2n∑1≤k≤nakbk−2(∑k=1nak)(∑k=1nbk)2n\sum_{1\le k\le n} a_k b_k - 2\left(\sum_{k=1}^n a_k\right)\left(\sum_{k=1}^n b_k\right) 2n1≤k≤n∑​ak​bk​−2(k=1∑n​ak​)(k=1∑n​bk​)
