新一代软机器与智能

讲师：刘子顺
日期：3.19

一、职业挑战？

  • 吊车问题 → 力学平衡计算

二、力学基础概念

  • 应力（Stress）：σ=FA\boldsymbol{\sigma = \frac{F}{A}}σ=AF​（类比“压强”，力/面积）
  • 应变（Strain）：ε=ΔLL0\boldsymbol{\varepsilon = \frac{\Delta L}{L_0}}ε=L0​ΔL​（形变与原长的比值）
  • 强度、刚度、稳定性：材料/结构的力学性能指标

三、人体力学与生物应用

  • 人体力学利用：高跟鞋问题（力学设计）、生物力学利用（如鼾声分析）
  • 结构强度案例：巴塞罗那大教堂（无直线，全砖建）→ 重力学

四、钢琴结构与材料

  • 钢琴结构：水凝胶（软机器）→ 硬机器（传统）→ 软机器（未来趋势）
  • 材料方向：智能软材料（未来三十年核心科技之一）

五、软材料与仿生科技

（1）候选材料

  • 水凝胶、介电弹性体

（2）特性与应用

  • 特性：弹性、人体机能模拟、外界交换
  • 高分子网络 + 水分子 → 凝固（如布丁、豆腐）
  • 隐形眼镜 → 人体软骨/半月板修复
  • 2003 龚剑萍：双网水凝胶（water: 90%，stretch: 21）→ 光照自动走
  • 负泊松比材料：越拉越粗（力学反常识）
  • 水凝胶模具、导线（可逆拉伸）、皮肤再生、半月板修复

六、模式转换（力学→功能）

  • 公式：□→△→□\square \rightarrow \triangle \rightarrow \square□→△→□（形状/性能切换）
  • 改变温度 → 表情随之改变（软机器+AI=机器人）

七、软物质力学（仿自然）

  • 叶序（花朵形状、瓜的形状）→ 成功力学解释
  • 叶片干的过程 → 力学
  • 花早开/晚开 → 温度调控
  • 苦瓜变形、苹果、仙人掌、青椒 → 形状记忆聚合物（不同温度、光、pH → 不同形状）

应用案例

  • 飞机改变翅膀（气动优化）
  • 卫星太阳能板打开（空间展开）
  • 花开、血管支架（生物医学）
