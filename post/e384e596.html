题目

将一个８*８的棋盘进行如下分割：将原棋盘割下一块矩形棋盘并使剩下部分也是矩形，再将剩下的部分继续如此分割，这样割了(n-1)次后，连同最后剩下的矩形棋盘共有n块矩形棋盘。(每次切割都只能沿着棋盘格子的边进行)

原棋盘上每一格有一个分值，一块矩形棋盘的总分为其所含各格分值之和。现在需要把棋盘按上述规则分割成n块矩形棋盘，并使各矩形棋盘总分的均方差最小。
均方差，其中平均值，xi为第i块矩形棋盘的总分。
请编程对给出的棋盘及n，求出O'的最小值。

思路

化简式子可知原式是求：

∑i=1nxi2−(∑i=1nxi)2nn\Large\sqrt{\frac{\sum_{i=1}^n x_i^2-\frac{(\sum_{i=1}^n x_i)^2}{n}}{n}} n∑i=1n​xi2​−n(∑i=1n​xi​)2​​​

(∑i=1nxi)2n\frac{(\sum_{i=1}^n x_i)^2}{n}n(∑i=1n​xi​)2​ 可以直接求出来，所以主要是弄前面一部分。

设 dp(lx,ly,rx,ry,k)dp(lx, ly, rx, ry, k)dp(lx,ly,rx,ry,k) 表示在 (lx,ly)(lx, ly)(lx,ly) 到 (rx,ry)(rx, ry)(rx,ry) 的区间里分成 kkk 块每块的平方和最小值。

转移时可以考虑横切竖切的情况。

为了方便统计答案，上面的式子 ∑i=1nxi2−(∑i=1nxi)2nn\sqrt{\frac{\sum_{i=1}^n x_i^2-\frac{(\sum_{i=1}^n x_i)^2}{n}}{n}}n∑i=1n​xi2​−n(∑i=1n​xi​)2​​​ 可以化简为：

∑i=1nxi2n−(∑i=1nxin)2\Large\sqrt{\frac{\sum_{i=1}^n x_i^2}{n}-(\frac{\sum_{i=1}^n x_i}{n})^2} n∑i=1n​xi2​​−(n∑i=1n​xi​​)2​

题目

Define the distance between two strings of the same length as the numbers of the positions where the characters differ in these two strings.

If two strings of the same length has a distance of no more than k, we call these two string satisfy k−matching.

Given a string S of length n and a integer k. For each i∈[1,n−1], split S into substrings A and B, while A=S[1,i] and B=S[i+1,n]. For all the string pairs formed by some non empty substring of A and some non empty substrings of B, count the numbers of pairs satisfying k−matching.

给定一个串 SSS，对于每一个前缀 AAA 以及对应的后缀 BBB，问有多少个子串对之间的距离不超过 K。
两个相同长度的串的距离定义为他们不同字符的个数。

len <= 3000, K <= 3000

思路

设 dp(i,j)dp(i, j)dp(i,j) 表示从 iii 和 jjj 开始的前缀中距离不超过 KKK 的最长长度。(假设 i<ji<ji<j)

然后我们考虑对答案的贡献。

然后我们会发现有当分割点不同时会有四种状态：

  • A：[1,i−1][1, i-1][1,i−1] 这个时候没有包括 iii，贡献为0
  • B：[i,i+dp(i,j)−1][i, i+dp(i,j)-1][i,i+dp(i,j)−1] 这个时候包括了 iii，且答案会逐渐递增。
  • C：[i+dp(i,j),j−1][i+dp(i, j), j-1][i+dp(i,j),j−1] 这个时候 iii 在一边，jjj 在一边，答案保持为 dp(i,j)dp(i, j)dp(i,j)
  • D：[j,n][j, n][j,n] 这个时候 iii 和 jjj 都在分割点的左边，贡献为0

然后我们尝试对答案数组差分

  • A：[1,i−1][1, i-1][1,i−1] 差分为0
  • B：[i,i+dp(i,j)−1][i, i+dp(i,j)-1][i,i+dp(i,j)−1] 差分为1
  • C：[i+dp(i,j),j−1][i+dp(i, j), j-1][i+dp(i,j),j−1] 差分为0
  • j: 差分为 −dp(i,j)-dp(i, j)−dp(i,j)
  • D：[j+1,n][j+1, n][j+1,n] 这个时候 iii 和 jjj 差分为0

然后我们再做一次差分：

  • iii：111
  • i+dp(i,j)i+dp(i,j)i+dp(i,j)：−1-1−1
  • jjj：dp(i,j)dp(i,j)dp(i,j)
  • j+1j+1j+1：−dp(i,j)-dp(i,j)−dp(i,j)

然后我们维护二次差分数组，最后统计答案时做两次前缀和即可。

至于 dp(i,j)dp(i,j)dp(i,j) 可以用双指针维护，甚至数组也不用开了。

时间复杂度：O(n2)O(n^2)O(n2)

题目

Your company has just constructed a new skyscraper, but you just noticed a terrible problem: there is only space to put one game room on each floor! The game rooms have not been furnished yet, so you can still decide which ones should be for table tennis and which ones should be for pool. There must be at least one game room of each type in the building.

Luckily, you know who will work where in this building (everyone has picked out offices). You know that there will be Ti table tennis players and Pi pool players on each floor. Our goal is to minimize the sum of distances for each employee to their nearest game room. The distance is the difference in floor numbers: 0 if an employee is on the same floor as a game room of their desired type, 1 if the nearest game room of the desired type is exactly one floor above or below the employee, and so on.

计划建设 N 层高的运动楼，每层楼要么建游泳池，要么建乒乓球室，只能二选一。
已知第 i 层有 TiT_iTi​ 个人打乒乓球，PiP_iPi​ 个人想游泳。
如果该层楼没有对应的功能室，对应的人只能去最近的楼层运动。
单个人爬一层楼的额外运动量是 1，问最合理的设计下，满足每个人的需求时，最少付出的额外运动量总和是多少。

N<=4000, 0<=Ti,Pi<=109N <= 4000,\ 0 <= T_i,P_i <= 10^{9}N<=4000, 0<=Ti​,Pi​<=109

设 dp(i,k)dp(i, k)dp(i,k) 表示第 iii 层修建设施 kkk，且第 i+1i+1i+1 层修建另一种建筑物，前 iii 层的最小代价。

然后我们枚举一个楼层 jjj，使 [j+1,i][j+1, i][j+1,i] 层都修建建筑 kkk，然后我们再让 [j+1,i][j+1, i][j+1,i] 不是 kkk 的人去 jjj 或 i+1i+1i+1 层去。

dp(i,k)=min⁡j=1i−1dp(j,k)+go(j+1,i,k′)\Large dp(i, k)=\min_{j=1}^{i-1}dp(j, k)+go(j+1, i, k') dp(i,k)=j=1mini−1​dp(j,k)+go(j+1,i,k′)

初始化我们假设前 iii 层都修建建筑 kkk。

注意在转移过程中我们就要统计答案，我们可以假设 [i+1,n][i+1, n][i+1,n] 都修建建筑 k′k'k′。

题目

There is a n×m board, a chess want to go to the position
(n,m) from the position (1,1).
The chess is able to go to position (x2,y2) from the position (x1,y1), only and if only x1,y1,x2,y2 is satisfied that (x2−x1)2+(y2−y1)2=5, x2>x1, y2>y1.
Unfortunately, there are some obstacles on the board. And the chess never can stay on the grid where has a obstacle.
I want you to tell me, There are how may ways the chess can achieve its goal.

问 N∗MN * MN∗M 的矩阵，从左上角到右下角的方案数。
其中：

  • 相邻两次的步长必须满足：(y2−y1)2+(x2−x1)2=5 && y2>y1 && x2>x1(y_2 - y_1)^2 + (x_2 - x_1)^2 = 5\ \&\&\ y2 > y1\ \&\&\ x2 > x1(y2​−y1​)2+(x2​−x1​)2=5 && y2>y1 && x2>x1；
  • 有 KKK 个格子不能进入。

N,M<=109, K<=100N,M <= 10^{9},\ K<=100N,M<=109, K<=100

思路

我们发现棋盘很大，不可能一个个格子算，于是我们尝试对关键点（不能到的格子和终点）来算。

引理：

在 n×mn\times mn×m 的棋盘中，每次可以向下或向右走一格，从左上角到右下角的方案为:

Cn+mn=Cn+mm\Large C_{n+m}^n=C_{n+m}^m Cn+mn​=Cn+mm​

然而这道题中是走日子格，于是我们可以吧棋盘变一下，变成上面的经典问题。

假设当前位置在 (i,j)(i, j)(i,j)，则从起点要走 (i+j−2)(i+j-2)(i+j−2) （这是当 i≠1,j≠1i\neq 1 ,j\neq 1i=1,j=1 时，不过数据好像没有卡，所以我代码也没写）个格子到这里，而日子格每次可以走3格，所以会走 (i+j−2)3\frac{(i+j-2)}{3}3(i+j−2)​ 步。每次走日子格都至少向下和向右一格，然后是向下或向右一格，所以 (i,j)(i, j)(i,j) 就可以转化为 (i−(i+j−2)3,j−(i+j−2)3)(i-\frac{(i+j-2)}{3}, j-\frac{(i+j-2)}{3})(i−3(i+j−2)​,j−3(i+j−2)​)。不合法的情况还需要特判一下。

设 dpidp_idpi​ 表示到达第 iii 个关键点的合法方案数。然后我们套容斥枚举第一个不合法的点，假设是 dpjdp_jdpj​，然后再拿乘上从 jjj 走到 iii 的方案数。

dpi=f(1,1,xi,yi)−∑j=1i−1(dpj×f(xj,yj,xi,yi))\Large dp_i=f(1, 1, x_i, y_i)-\sum_{j=1}^{i-1}(dp_j\times f(x_j, y_j, x_i, y_i)) dpi​=f(1,1,xi​,yi​)−j=1∑i−1​(dpj​×f(xj​,yj​,xi​,yi​))

题目

Bubu’s bookshelf is in a mess! Help him!
There are nbooks on his bookshelf. We define the mess value to be the number of segments of
consecutive equal-height books. For example, if the book heights are 30, 30, 31, 31, 32, the mess
value is 3, that of 30, 32, 32, 31 is also 3, but the mess value of 31, 32, 31, 32, 31 is 5 — it’s indeed in
a mess!
Bubu wants to reduce the mess value as much as possible, but he’s a little bit tired, so he decided
to take out at most kbooks, then put them back somewhere in the shelf. Could you help him?

书架上有 N 本书，书本高度是 [25cm，32cm]，凌乱地排着。
定义一排书的整齐度是指最少可以被划分成高度连续相等的书的段数。
比如 {30，32，32，31} 整齐度是 3。
比如 {31，32，31，32，31} 整齐度是 5。
现在最多从其中拿出 K 本书，然后塞回去。
问整理后，最少可以有多少段高度相等的连续区间。

K <= N <= 100

思路

发现书本高度最大最小差为8，很容易想到状压dp。

设 dp(i,j,k,o)dp(i, j, k, o)dp(i,j,k,o) 表示前 iii 本书里拿出 jjj 本，剩下书高度集合为 kkk，且最后一本书高度为 ooo 的最小凌乱度。

然后枚举每一本书拿或者不拿。

如果不拿，则 kkk 变成 k∣(1<<ai)k|(1<<a_i)k∣(1<<ai​)，代表这本书放入未被拿的集合。ooo 变成 aia_iai​。如果之前 o≠aio\neq a_io=ai​，则要加1。

如果拿，剩下书的集合还有最后一本不变。然后我们考虑是否+1，如果为拿出的集合中有这本书，我们可以把现在这本书插到它旁边，然后不用+1。如果后面还有和它一样的书，我们就把现在这本书的命运交给后面那本书。除了这两种情况，其它都要+1。

可能会MLE，要滚动数组优化。
