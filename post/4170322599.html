List

Creating Lists

Note : type homogeneous

  • [1,2,3]
  • [1..5]、[2,4..10]

Accessing List Elements

  • index from 0
  • !!：list !! index

Concatenation & Extension

  • Concatenation : ++ like [1, 2, 3] ++ [4, 5, 6]
  • Cons operator: : like 0 : [1, 2, 3] and we get [0, 1, 2, 3]

List Slicing

  • take : take 3 [1, 2, 3, 4, 5] is [1, 2, 3]
  • drop : drop 2 [1, 2, 3, 4, 5] is [3, 4, 5]
  • splitAt: splitAt 2 [1, 2, 3, 4, 5] is [1, 2], [3, 4, 5], which returns a tuple
