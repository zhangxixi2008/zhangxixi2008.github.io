题目

There is a n×m board, a chess want to go to the position
(n,m) from the position (1,1).
The chess is able to go to position (x2,y2) from the position (x1,y1), only and if only x1,y1,x2,y2 is satisfied that (x2−x1)2+(y2−y1)2=5, x2>x1, y2>y1.
Unfortunately, there are some obstacles on the board. And the chess never can stay on the grid where has a obstacle.
I want you to tell me, There are how may ways the chess can achieve its goal.

问 N∗MN * MN∗M 的矩阵，从左上角到右下角的方案数。
其中：

  • 相邻两次的步长必须满足：(y2−y1)2+(x2−x1)2=5 && y2>y1 && x2>x1(y_2 - y_1)^2 + (x_2 - x_1)^2 = 5\ \&\&\ y2 > y1\ \&\&\ x2 > x1(y2​−y1​)2+(x2​−x1​)2=5 && y2>y1 && x2>x1；
  • 有 KKK 个格子不能进入。

N,M<=109, K<=100N,M <= 10^{9},\ K<=100N,M<=109, K<=100

思路

我们发现棋盘很大，不可能一个个格子算，于是我们尝试对关键点（不能到的格子和终点）来算。

引理：

在 n×mn\times mn×m 的棋盘中，每次可以向下或向右走一格，从左上角到右下角的方案为:

Cn+mn=Cn+mm\Large C_{n+m}^n=C_{n+m}^m Cn+mn​=Cn+mm​

然而这道题中是走日子格，于是我们可以吧棋盘变一下，变成上面的经典问题。

假设当前位置在 (i,j)(i, j)(i,j)，则从起点要走 (i+j−2)(i+j-2)(i+j−2) （这是当 i≠1,j≠1i\neq 1 ,j\neq 1i=1,j=1 时，不过数据好像没有卡，所以我代码也没写）个格子到这里，而日子格每次可以走3格，所以会走 (i+j−2)3\frac{(i+j-2)}{3}3(i+j−2)​ 步。每次走日子格都至少向下和向右一格，然后是向下或向右一格，所以 (i,j)(i, j)(i,j) 就可以转化为 (i−(i+j−2)3,j−(i+j−2)3)(i-\frac{(i+j-2)}{3}, j-\frac{(i+j-2)}{3})(i−3(i+j−2)​,j−3(i+j−2)​)。不合法的情况还需要特判一下。

设 dpidp_idpi​ 表示到达第 iii 个关键点的合法方案数。然后我们套容斥枚举第一个不合法的点，假设是 dpjdp_jdpj​，然后再拿乘上从 jjj 走到 iii 的方案数。

dpi=f(1,1,xi,yi)−∑j=1i−1(dpj×f(xj,yj,xi,yi))\Large dp_i=f(1, 1, x_i, y_i)-\sum_{j=1}^{i-1}(dp_j\times f(x_j, y_j, x_i, y_i)) dpi​=f(1,1,xi​,yi​)−j=1∑i−1​(dpj​×f(xj​,yj​,xi​,yi​))

组合数可以用lacus定理来优化。
