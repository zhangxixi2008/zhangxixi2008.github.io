题目

给定两个整数 l 和 r ，对于所有满足1 ≤ l ≤ x ≤ r ≤ 10^9 的 x ，把 x 的所有约数全部写下来。对于每个写下来的数，只保留最高位的那个数码。求1～9每个数码出现的次数。

思路

显然数论分块

然后统计一下每一块内1到9出现的情况乘上 n/ln/ln/l 即可

Code

总结

首先看到这道题，把式子列出来，很容易发现是数论分块

然后每个数出现的个数想一下就打了
