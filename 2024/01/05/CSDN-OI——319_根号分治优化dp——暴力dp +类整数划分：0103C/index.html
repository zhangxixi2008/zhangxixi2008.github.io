看到这种和达到平方级别，比如初项和长度成反比的题目，应该要想到根号分治

我们随便分，比如初项 ≤B\le B≤B ，那 jjj 不超过 2B2B2B ，直接跑即可。

如果 >B>B>B ，我们考虑类整数划分思路。 g(i,j)g(i,j)g(i,j) 表示现在到第 iii 个，所有数总和为 jjj ，且 [i,n][i,n][i,n] 和 i−1i-1i−1 目前相同。则现在对于 iii ，我们可能±1，为了保证正确性，我们让后面的整体±1，也就是转移到 g(i+1,j+i),g(i+1,j−i)g(i+1,j+i),g(i+1,j-i)g(i+1,j+i),g(i+1,j−i) 。显然长度不超过 O(nB)O(\frac n B) O(Bn​) 。

空间方面，第二个直接滚。第一个发现 iii 可以按  mod  2B\bmod\, 2Bmod2B 的余数分类。因此空间是线性的。
