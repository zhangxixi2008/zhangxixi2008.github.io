http://47.92.197.167:5283/contest/441/problem/3

首先有个暴力dp。 $f(i,j)$ 和为 $i$ ，末项为 $j$ 。

看到这种和达到平方级别，比如初项和长度成反比的题目，应该要想到根号分治

我们随便分，比如初项 $\le B$ ，那 $j$ 不超过 $2B$ ，直接跑即可。

如果 $>B$ ，我们考虑类整数划分思路。 $g(i,j)$ 表示现在到第 $i$ 个，所有数总和为 $j$ ，且 $[i,n]$ 和 $i-1$ 目前相同。则现在对于 $i$ ，我们可能±1，为了保证正确性，我们让后面的整体±1，也就是转移到 $g(i+1,j+i),g(i+1,j-i)$ 。显然长度不超过 $O(\frac n B)
$ 。

空间方面，第二个直接滚。第一个发现 $i$ 可以按 $\bmod, 2B$ 的余数分类。因此空间是线性的。

然后我被卡常了。
